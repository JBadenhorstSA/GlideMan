Glider Charge Options Explanation

This is an explanation of the different charging options for gliders and the formula that needs to be entered for each to perform the calculations

Option 1:

Charges the flight according to a normal hourly charge. Takes the flight duration multiplied by the charge per hour

FORMULA : HourCharge e.g. 100


Option 2:

Uses a per minute charge with minimum and maximum per flight charge. If the flight cost is less that the minimum, the minimum charge is applied, if the flight is more than the maximum , the maximum flight charge is applied otherwise the flight cost is calculated on the per minute rate

FORMULA : PerMinuteCost-MinCost-MaxCost e.g 5-300-500


Option 3:

Uses a minimum flight cost and maximum flight cost. There is a minimum cost for each flight and if the flight is longer than a given period of time tha maximum charge is applied. The time period is specified in decimal as a fraction of an hour or decimal as ammount of hours. e.g 30 minutes is specified as 0.5

FORMULA : MinCost-MaxCost-TimeSplit e.g 50-100-0.5


TMG Option 1 :

Only charges for the amount of time the engine was running using the tacho time. It then also adds the cost for each landing given the cost per landing ammount

FORMULA : TachoHourCharge-LandingCharge e.g. 650-10


TMG Option 2:

The same as for Option 1 except that it also charges a rate for the engine off time (Gliding Time)

FORMULA : GlidingHourCharge-TachoHourCharge-LandingCharge e.g 150-650-10
